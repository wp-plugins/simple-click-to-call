Sistema para contactarse con un agente de tu sitio.

== Description ==

Simple Click to Call es un pequeño formulario en tu sitio web que permite que los usuarios se contacten con vos en forma inmediata realizando una llamada de teléfono a teléfono sin necesidad de tener que discar números.

Podrás ayudar a tus futuros clientes en la decisión de contratar los servicios ofrecidos o darle respuesta a cualquier inquietud.

== Installation ==

1. Unzip `simple-click-to-call.zip` in the `/wp-content/plugins/` directory, making sure the folder is called `simple-click-to-call`.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Visit the admin page `Click To Call` and save your preferences.
